# NAPE Docs

## Build Variables

Below are the variable names that are being stored in the GitHub Actions variables, as opposed to directly commited to the repo.

### Environment Variables

- `INSTNACE`
- `ARTIFACT`
- `DOCKER_VERSION`
- `CONFIG_JSON_PRODUCT`
- `CONFIG_JSON_VERSION`

### Secrets

- `ALGOLIA_ARTIFACT`
- `ALGOLIA_APP_NAME`
- `ALGOLIA_INDEX_NAME`
- `ALGOLIA_KEY`
